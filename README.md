## A simple C program that simulates a calculator (Mathematical & Bitwise)


### Features

* Supports mathematical operations:
  - Addition (+)
  - Subtraction (-)
  - Multiplication (*)
  - Division (/)
  - Power (^)
  - Root (r)

* Supports bitwise operations:
  - AND (&)
  - OR (|)
  - XOR (^)
  - NOT (~)
  - Left Shift (<<)
  - Right Shift (>>)

...
## How to Run

1. Clone this project

2. Open the folder in terminal

3. Compile the program:

-> gcc *.c -o main -lm

4. Run the program:

-> ./main
